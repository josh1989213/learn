常見置中方式

  1. margin: 0 auto
  2. inline ; inline-block 的行內置中方式
  3. flex置中
  4. position:absolute的置中
  5. 文字的置中

margin: 0 auto

必須是區塊的元素

display必須是blocl或是flex才能用這種方式置中

寬度必須是定值

只能用在水平置中，不能垂直，較常用於網頁中較大的元素的排版

第二種寫法margin-right:auto; margin-left:auto;

讓我去中間

行內元素置中方式

inline ; inline-block

需要同時控制內層跟外層

外層寫上text-align: center;內層的元素被當成文字置中了

內層元素必須是inline或是inline-block

此時內層繼承了上層屬性所以裡面文字也置中了 如果不想要文字置中可在內層box中寫text-align: right等改寫屬性

這種置中方式常用於文字的置中

讓我去中間

彈性排版flex的置中方式

外層寫上display:flex ;內層寫margin:auto

這種方式比margin: 0 auto更加靈活而且可以做到水平跟垂直的置中,上下左右置中

需要水平置中就寫margin: 0 auto

使用justify-content: center;讓框框置中

align-items: center;垂直置中

垂直置中也可在內層寫align-self: center，這時候外層就不用align-items: center了
