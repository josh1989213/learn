動畫屬性以及關鍵影格

小小筆記

螺旋完跟千鳥是只是示範動畫方向，跟三個動畫元素的互動效果，三個動畫的秒數之間配合
  1. animation-name 動畫名稱
  2. @keyframes 定義動畫要寫這個，關鍵影格的意思
  3. 寫法是@keyframes+動畫名稱{.... }
  4. 關鍵影格0~100格可任意抓取，本示範抓取兩個影格
  5. 定義影格內屬性的寫法就是 @keyframes 向右{ 0% left: 30%; } 100%{ left: 70%; } }
  6. animation-duration: 3s; 動畫時間3秒鐘的意思
  7. animation-delay: 1s 動畫延遲1秒撥放
  8. animation-iteration-count: 3 就是撥放三次的意思
  9. animation-iteration-count:infinite 無限播放
  10. animation-direction: reverse; 動畫倒轉
  11. animation-timing-function動畫的貝茲曲線，跟"轉場效果"的一樣
  12. animation-fill-mode:，『動畫填充模式』決定動畫開始或結束的停留狀態，
    分別為三個值:backwards 「開始前的狀態」、forwards「開始前的狀態」
  13. 舉例寫上 animation-fill-mode: backwards;時，動畫會停在0%影格的left: 30%;
  14. animation-fill-mode: both;動畫填充模式，同時選擇倆著，
    同時定義開始跟停止的狀態。
