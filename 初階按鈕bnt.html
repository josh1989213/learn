小小筆記

  • css中的*{}代表選擇所有的標籤，所有標籤裡都這樣設定的意思
  • *這個選擇器是權重最低的
  • *{ position: relative; }
    這是設計師們很常用的方法
  • 關於歸零，這裡沒有呼叫reset.css所以用html,body{ padding: 0; margin: 0; height: 100%;}來做歸零，
    height高度撐開100% 依據視窗大小，如果沒有寫的話就是按造內容大小了 }
  • 使用padding而不是固定寬高可以讓外框自動調整，可以隨意增加字數
  • 當border-radius:是方框的一半(line-heghit+padding=44px)時就會變成橢圓囉
  • 當寫上 position:absolut的時候就會脫離display的框架了，所以把display: inline-block註解掉
  • 在.bnt:before的圓形是這樣做的
    width: 100% padding-top: 100%
    width的100%依據來自於a連結的寬度，padding-top(內距-上)的100%依據是width的寬度，所以會呈現1:1的數字，變成正方形，不能直接用height: 100%是因為這個100依據來自a連結的寬 出來結果不會是1:1的
    最後加個border-radius: 50%;就變成正圓形了
  • overflow: hidden; 隱藏多餘的部分
  • box-shadow: 0 10px 5px rgba(90, 93, 93, 0.5);值的說明如下 x軸 y軸 模糊值 rgba
